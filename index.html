SHC Application Framework

Das Application Framework ist der Nachfolger des Raspberry Pi SmartHome Control 2.x. Das Apllication Framework wird zu großen Teilen in Java geschrieben und bricht damit einige grenzen des in PHP geschriebenen RPi SmartHome Controls auf.

Das Framework besteht aus mehreren Modulen:

  • SHC Core -> eine Libary welche die Datentypen vorgibt und die Basis für alle anderen Module Bildet
  • SHC Application Server -> ist die Wichtigste Komponente, der Server übernimmt die Datenhaltung und Verwaltung sowie die Berechnung der Automatisierungsaufgaben
  • SHC CLI Client -> ist die Kommandozeilen Schnittstelle (nur Schalten und Status Abfragen)
  • SHC Desktop Client -> ist die neue Kernoberfläche, der Desktop Client wird in javaFX geschrieben und stellt alle Funktionen der Benutzeroberfläche für den App Server bereit
  • SHC Android App -> die Android App wird ähnlichen Funktionsumfang wie der Desktop Client bekommen und damit auch alle Verwaltungsfunktionen
  • SHC Web Client -> der Web Client wird eine Desktop und Mobil Ansicht bekommen aber nur das Schalten und abfragen des Status ermöglichen
  • SHC Switchserver -> der Schaltserver ist die Schnittstelle zu den Hardware Funktionen des Raspberry Pi und übernimmt z.B.: das Senden der 433MHz Kommandos
  • SHC Sensortransmitter -> der Sensorsender schickt die Sensordaten des Raspberry Pi an den App Server

Unterstützte Funktionen

  • schaltbare/lesbare Elemente
    • 433MHz Steckdosen
    • AVM DECT Steckdosen
    • Edimax SP-1101W und SP2101W WLan Steckdosen neu
    • Fritz!Box Funktionen (WLan an/aus, Neustart ...)
    • Neustart/Herunterfahren des Servers
    • Neustart/Herunterfahren von Schaltservern neu
    • GPIOs vom Raspberry Pi und Microcontrollern als Eingang lesen und als Ausgang schalten
    • Scripte auf dem App Server oder Schaltserbern neu ausführen
    • Rechner im Netzwerk starten "Wake on Lan"
  • Gruppen Elemente
    • Aktivitäten (Gruppenschaltung von mehreren schaltbaren Elementen)
    • Countdown (zeitlich begenzte Gruppenschaltung von mehreren schaltbaren Elementen)
    • Button (Tastfunktion)
  • Sensoren
    • DS12x20 Sensoren
    • DHT Sensoren
    • BMP Sensoren
    • Regen Sensoren
    • Feuchtigkeits Sensoren
    • Lichtstärke Sensoren
    • Wasserzähler, Gaszähler Stromzähler neu
    • HC-SR04 Sensoren neu
    • SCT-013 Sensoren neu
    • Akkus (Ladezustand) neu
  • virtuelle Sensoren (Statistische Mittelwerte von Sensoren mit gleichen Werten)
    • Energiemesser neu
    • Stromverbrauch neu
    • Feuchtigkeit neu
    • Lichtstärke neu
    • Temperatur neu
    • Luftfeuchte neu
    • Menge (Strom, Gas, Wasser) neu
